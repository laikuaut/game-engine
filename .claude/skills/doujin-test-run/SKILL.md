# doujin-test-run — テスト実行スキル

## 手順

1. `doujin-engine/` ディレクトリで `npx vitest run` を実行
2. 結果を確認:
   - **全パス** → 完了
   - **失敗あり** → 失敗内容を分析し修正、再実行（最大2回リトライ）
   - **3回失敗** → ユーザーに報告し、原因と対処案を提示

## 出力

```
## テスト結果

- ファイル数: X
- パス: X件
- 失敗: X件
- リトライ: X回
```
